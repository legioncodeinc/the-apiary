# Architecture Decision Records (umbrella)

> Category: Index | Version: 1.0 | Date: July 2026 | Status: Active

ADRs for cross-repo / superproject-level decisions in The Apiary (the combined release train, the installer/product-loading model, fleet-wide telemetry transport, the neutral fleet state root, the proposed fleet-shared embedding daemon). Submodule-scoped decisions live in each submodule's own `library/knowledge/private/architecture/`.

Each ADR is `ADR-<NNNN>-<slug>.md`, Nygard format, repo-local sequential numbering.
