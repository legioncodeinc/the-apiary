# live-chat-support-stinger

The operational playbook for `live-chat-support-worker-bee` — the Legion AI Army specialist for customer support widget integration, identity verification, conversation routing, AI deflection, and data-export discipline.

## What this stinger covers

- **Platform selection** — Plain, Pylon, Intercom, Crisp, Help Scout; decision matrix calibrated for B2B SaaS, PLG, and bootstrapped teams. Drift is sunset (March 2026) and excluded.
- **Widget integration** — JS snippet, React SDK, Next.js App Router patterns, CSP header configuration.
- **Identity verification** — HMAC-SHA256 and JWT signing (server-side only), key rotation, replay prevention.
- **Conversation routing** — team, skills-based, round-robin, overflow, priority queues for paying customers.
- **AI deflection** — Intercom Fin 2.0 (51% resolution, $0.99/resolution), Plain Ari (included every plan), Crisp Bot; handoff rules and knowledge base seeding.
- **Data export** — GDPR Article 20 portability, platform export paths, day-1 checklist.
