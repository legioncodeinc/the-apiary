# Reports

This folder accumulates integration audit reports produced by `live-chat-support-worker-bee` over time.

## Naming convention

`YYYY-MM-DD-<product>-<platform>-audit.md`

Example: `2026-06-01-acme-intercom-audit.md`

## What goes here

Reports generated when the Bee is asked to "audit this integration" or "review our live chat setup." Each report should be the completed `templates/platform-audit.md` with findings filled in.

## This folder is empty on first install

Audit reports are produced on demand. There are no pre-seeded reports.
