# Reports

This folder accumulates per-run submission audit logs and submission-readiness reports produced by `app-store-submission-worker-bee`.

Each report should be named with the date and app name:
```
YYYY-MM-DD-<app-name>-submission-report.md
```

Use `../templates/submission-readiness-report.md` as the starting template for each report.

The folder is initially empty. Reports are appended by the Bee during active engagements.
