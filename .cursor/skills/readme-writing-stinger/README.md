# readme-writing-stinger

The procedural arsenal for `readme-writing-worker-bee`. Encodes the structural rules, badge discipline, OSS/internal distinction, and README-driven development methodology that turn a README into a conversion surface.

This stinger was forged from `ai-tools/command-briefs/readme-writing-worker-bee-command-brief.md` and the research gathered by `scripture-historian` in `research/research-summary.md`.

The foundational constraint: **a visitor makes a go/no-go decision in 30 seconds.** Every guide in this folder derives from that constraint.
