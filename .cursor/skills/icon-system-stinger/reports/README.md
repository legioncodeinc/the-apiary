# Reports — icon-system-stinger

This folder collects past icon audit reports produced by `icon-system-worker-bee`. Each file represents one audit run, named by date and branch/PR.

**Naming convention:** `YYYY-MM-DD-<branch-or-pr>-icon-audit.md`

Reports are generated from `templates/icon-audit-report.md`. This folder starts empty; reports accumulate over time as the Bee is invoked.
