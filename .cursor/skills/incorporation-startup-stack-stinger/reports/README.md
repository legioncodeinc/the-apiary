# Reports

This folder collects past formation-report artifacts produced by `incorporation-startup-stack-worker-bee` over time.

Each completed engagement may optionally produce a saved report at:
`docs/formation/formation-report-<YYYY-MM-DD>.md`

The template for this report lives at `templates/formation-decision-report.md`.

**This folder is initially empty.** Reports accumulate as the Bee is used.
