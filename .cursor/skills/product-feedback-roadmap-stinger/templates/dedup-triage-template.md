## Pre-session checklist (5 min before)

- [ ] Export or pull the last 7 days of new submissions.
- [ ] Open the AI duplicate suggestions queue (if available on your platform).
- [ ] Have the canonical tag taxonomy open: [link to taxonomy doc].

---

## Agenda

### 1. New submissions review (10 min)

For each submission in the last 7 days:

- [ ] Is this a **duplicate** of an existing request? → Merge.
- [ ] Is this a **refinement** of an existing request? → Update the canonical request's description; close this one.
- [ ] Is this **new**? → Assign a primary tag; leave in "Under Review."

### 2. AI duplicate suggestions review (5 min)

If the platform surfaces AI-detected duplicates:

- Review each suggestion.
- Confirm or reject.
- Merge confirmed duplicates immediately.

**AI suggestions confirmed and merged:** _____

### 3. 30-day SLA backlog (10 min)

Run the "Under Review for > 30 days" filter.

For each item in backlog:

- [ ] Decision: **Planned** (commit to roadmap) or **Not Planned** (decline with reason)?
- If Planned: assign target milestone internally; trigger customer notification.
- If Not Planned: write the reason (1-3 sentences); trigger customer notification.
