# Example: Platform Recommendation — Newsletter-First Creator

*This example demonstrates `guides/00-platform-selection.md` Use Case A.*

## Scenario

User says: "I want to start a newsletter about productivity tools and eventually make money from it. I'm starting from zero subscribers. Which platform should I use?"

## Bee's response pattern

**Classify the use case first.** The user wants to build an audience as a business and monetize it directly (ads + paid subscriptions), starting from zero. This is Use Case A: newsletter-first.

**Recommendation: Beehiiv**

Start on Beehiiv's free plan (up to 2,500 subscribers) and upgrade to Scale ($49/mo) at 1,000 subscribers to unlock the Ad Network.

**Why Beehiiv specifically:**
- Ad Network: at 1,000 subscribers with a 40% open rate, you can start earning passive ad revenue. At $20 CPM on 400 opens/issue: ~$32/issue, ~$130/month from a weekly newsletter.
- Boosts: earn $1-$3 per subscriber you refer to other newsletters — zero writing required.
- 0% transaction fee on paid subscriptions (vs Substack's 10%, Kit's 0.6%).
- Referral program to grow the list organically.
- 66 days median to first dollar, per Beehiiv's 2026 platform data.

**What Beehiiv does not have:**
- Complex automation sequences (it has basics but Kit is better for elaborate drip funnels)
- Shopify/WooCommerce integrations (Kit has 100+; Beehiiv has essentially Stripe only)
- The best option if the user expects to sell courses or ebooks — Kit would be better for that

**Concrete next steps:**
1. Sign up at beehiiv.com — free plan, no credit card.
2. Verify a custom sending domain immediately (Beehiiv Settings → Sender Identity → Custom Domain).
3. Write the first 3 issues before building the sign-up form — have content ready before growing.
4. At 1,000 subscribers: upgrade to Scale and apply for Ad Network.
