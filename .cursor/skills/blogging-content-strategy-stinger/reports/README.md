# Reports

This folder collects past content strategy session outputs: cluster maps, post-brief sets, pre-publish checklist logs, and cadence plans produced by `blogging-content-strategy-worker-bee`.

## Naming convention

```
YYYY-MM-DD-<domain-or-client>-<session-type>.md
```

Examples:
- `2026-05-20-acme-saas-cluster-map.md`
- `2026-06-01-acme-saas-post-briefs-sprint-planning-cluster.md`
- `2026-06-15-acme-saas-checklist-log.md`

## Session types

| Type | Description |
|---|---|
| `cluster-map` | Full cluster architecture document (use `templates/cluster-map-template.md`) |
| `post-briefs` | Set of post briefs for a cluster (use `templates/post-brief-template.md` per post) |
| `checklist-log` | Pre-publish checklist run results for one or more posts |
| `cadence-plan` | Publishing cadence plan with capacity model |
| `blog-audit` | Retroactive audit of existing blog content (see `examples/existing-blog-audit.md`) |

## Notes

- Reports are produced outputs; do not edit them after delivery unless correcting a factual error.
- Maintain the checklist log as an ongoing document; add a row for each post reviewed rather than creating a new file per post.
