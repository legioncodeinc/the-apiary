## Maintenance Retainer (15-25% of build cost per year is standard)

| Build Cost | Annual Maintenance (15-25%) | Monthly |
|---|---|---|
| $50,000 | $7,500–$12,500 | $625–$1,042 |
| $100,000 | $15,000–$25,000 | $1,250–$2,083 |
| $160,000 | $24,000–$40,000 | $2,000–$3,333 |
| $200,000 | $30,000–$50,000 | $2,500–$4,167 |

**For a $160,000 build like Example Booking Co., fair maintenance is $2,000–$3,333/month.** DevPipe's $4,000–$6,000/month is 1.5–3× this range.

In hours: industry-standard maintenance for an app of this size is 20–30 hrs/month at $100/hr. DevPipe claimed 80 hrs/month.
