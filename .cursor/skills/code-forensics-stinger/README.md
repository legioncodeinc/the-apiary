# code-forensics-stinger

A Cursor IDE skill that equips the `code-forensics-worker-bee` Bee with the complete methodology for forensic investigations of software-development and agency-services engagements. Produces an 11-deliverable evidence packet (master forensic report, agency subreport, attorney legal memo, plain-language client report, 51-tab Excel workbook, and 6-document pre-litigation pack) ready for retained counsel.

This Stinger is the generic, parameterized descendant of `ada-devpipe-forensics`. It strips out specific defendant context (ADA/Robert Hartwell, DevPipe/Sameer Khan) and replaces it with pluggable defendant-profile and jurisdiction templates. The Example Booking Co. matter remains as a reference example.
