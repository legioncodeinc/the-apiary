# Reports

This folder collects archived Build Reports from past `website-worker-bee` runs. The Bee writes the *active* report into the target repo at `<repo-root>/build-report.md`; copies of completed reports may be checked back into this folder under `reports/<YYYY-MM-DD>-<site-name>.md` for historical reference.

The active template lives at `templates/build-report.md`. Do not edit completed reports; instead, generate a new run.
