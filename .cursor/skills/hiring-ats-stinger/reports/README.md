# Reports

This folder collects past ATS audit reports produced by `hiring-ats-worker-bee`.

Reports are produced using `templates/ats-audit-report.md`. Each report file should be named:

```
{YYYY-MM-DD}-{org-slug}-ats-audit.md
```

Example: `2026-05-20-acme-corp-ats-audit.md`

New reports are added here over time; existing reports are not modified after creation (they are point-in-time audit artifacts).
