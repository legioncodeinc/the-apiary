# Reports

This folder collects past-run audit reports from `telegram-bot-worker-bee`.

Each report is a dated markdown file named `YYYY-MM-DD-<scope>-report.md`. Reports are created by the Bee when it completes a significant review or implementation task (e.g., a webhook debugging session, a payments integration audit, a Mini Apps security review).

Reports are not created automatically; the engineer or the Bee may create one after a complex engagement.
