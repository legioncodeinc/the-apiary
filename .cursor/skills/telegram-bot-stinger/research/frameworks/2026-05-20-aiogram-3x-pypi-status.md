# aiogram v3.28.2 - PyPI Package Status

## Summary

aiogram is the leading async Python framework for Telegram bots. As of May 2026, the latest version is v3.28.2 (released May 10, 2026). The package explicitly states "Supports Telegram Bot API 10.0" - the very latest Bot API. It requires Python 3.10+ and uses asyncio + aiohttp. The GitHub repo has 5,640 stars and is on the `dev-3.x` branch. The v3.x release cadence is rapid: 5 releases in 2026 alone (3.24.0 in January through 3.28.2 in May). The framework has built-in FSM (Finite State Machine), magic filters, routers (Blueprints), middleware, and I18n/L10n support.
