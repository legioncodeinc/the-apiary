# Template: Option Grant Pre-Checklist

Complete ALL items before issuing any option grant. Missing any item can expose employees to tax penalties or create cap table errors that require retroactive correction.

Reference: `guides/04-409a-valuations.md`, `guides/05-option-pool-management.md`, `guides/06-vesting-schedules.md`

---

## For each new option grant, confirm:

### 409A validity
- [ ] A current 409A valuation report exists.
- [ ] The 409A report is within its 12-month validity window (valuation date, not delivery date).
- [ ] No material event has occurred since the last 409A (see `guides/04-409a-valuations.md` trigger list).
  - [ ] No signed term sheet since the last 409A.
  - [ ] No acquisition LOI since the last 409A.
  - [ ] No significant material business change since the last 409A.
- [ ] Strike price for this grant is set at or above the FMV in the current 409A report.

### Option pool availability
- [ ] Sufficient unissued options exist in the pool to cover this grant.
  - Available unissued options: [ENTER CURRENT AVAILABLE OPTIONS]
  - This grant size: [ENTER GRANT SIZE]
  - Remaining after grant: [AVAILABLE - GRANT SIZE]
- [ ] If pool is insufficient, board has approved a pool expansion before this grant.

### Board approval
- [ ] Board consent approving this specific grant has been prepared.
- [ ] Board consent includes: grantee name, number of options, grant date, strike price, vesting schedule.
- [ ] Board consent is signed by all required directors (check bylaws for quorum requirements).

### Grant type
- [ ] ISO or NSO designation is determined:
  - For US employees: ISO up to the $100K annual vesting limit, NSO beyond.
  - For contractors, advisors, international employees: NSO only.
- [ ] If ISO: confirm grantee is a US-based W-2 employee (not contractor).

### Vesting schedule
- [ ] Vesting schedule is documented in the grant agreement (4-year/1-year cliff or board-approved variation).
- [ ] Vesting commencement date is confirmed (usually hire date or a specific agreed date).
- [ ] Any acceleration provisions are explicitly documented (double-trigger, single-trigger, or none).

### Grant agreement
- [ ] Option grant agreement generated from the current plan template (same version as the board-approved plan).
- [ ] Grant agreement countersigned by authorized company signatory.
- [ ] Grantee has signed the grant agreement.
- [ ] Early exercise right (83(b) election option) is included if applicable.

### Cap table update
- [ ] Cap table platform (Carta / Pulley) is updated with the new grant:
  - Grantee name and contact information
  - Grant date (must match board consent date)
  - Number of options
  - Strike price
  - Vesting schedule
  - ISO/NSO designation
- [ ] Confirmation email sent to grantee with grant summary.

---

## After grant is complete

- [ ] Store signed grant agreement in the HR folder of the data room.
- [ ] Store board consent in the Corporate Legal folder of the data room.
- [ ] Confirm total outstanding options + unissued pool does not exceed authorized pool size.
- [ ] Flag on calendar: next 409A expiration date.
