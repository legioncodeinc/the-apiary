# Reports

This folder collects advisory reports produced by `investor-cap-table-worker-bee` over time.

Each report is a markdown file named `YYYY-MM-DD-<topic>-<company-or-context>.md`. Reports may include:
- Cap table audit summaries
- Platform migration recommendations
- Pre-raise dilution models
- Data room gap analyses
- Option pool refresh analyses

This folder is initially empty. Reports accumulate as `investor-cap-table-worker-bee` is invoked and the user opts to save an advisory output.
