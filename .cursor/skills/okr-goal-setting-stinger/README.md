# okr-goal-setting-stinger

The OKR methodology arsenal for `okr-goal-setting-worker-bee`.

This stinger encodes the Grove/Doerr OKR canon, the output-vs-input Key Result discipline, aspirational-vs.-committed calibration, quarterly cadence playbooks, OKR/KPI/MBO comparison, small-team adaptation, and tool configuration guides for Lattice, 15Five, Weekdone, and Notion.
