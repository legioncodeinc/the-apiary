# Reports

This folder collects past OKR audit reports and cycle summaries produced by `okr-goal-setting-worker-bee`.

## Naming convention

```
<date>-<team>-<cycle>-<report-type>.md
```

Examples:
- `2026-07-15-product-team-Q2-audit.md`
- `2026-10-01-company-Q3-retrospective-summary.md`
- `2026-04-01-engineering-Q1-calibration-review.md`

## Report types

| Type | Template | Description |
|---|---|---|
| OKR audit | `templates/okr-audit-report.md` | Health check on a team's OKRs; produced mid-cycle or at end of cycle |
| Retrospective summary | `templates/okr-retrospective.md` | End-of-quarter scoring session notes |
| Coaching session | Inline | Documented OKR rewrites from a coaching engagement |
