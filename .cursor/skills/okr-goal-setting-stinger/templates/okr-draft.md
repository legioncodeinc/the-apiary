## Pre-kickoff checklist

- [ ] Objective is aspirational, qualitative, and time-bound
- [ ] All KRs are output metrics (not milestones or activity counts)
- [ ] All KRs have a measured baseline (not estimated)
- [ ] All KRs have a concrete target and unit
- [ ] Each KR is classified as aspirational or committed
- [ ] No KR is linked to compensation
- [ ] The team has agreed on the data source for each KR
- [ ] OKR type (company / team / individual) is documented
- [ ] If team/individual: parent OKR is linked
