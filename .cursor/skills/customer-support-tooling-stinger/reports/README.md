# Reports

This folder accumulates past support audit reports produced by `customer-support-tooling-worker-bee` using the `templates/support-audit-report.md` skeleton.

Each report is named `YYYY-MM-DD-<product-name>-support-audit.md`.

Reports are written here when the Bee is invoked to audit an existing support stack. They are consumed by engineering leads, founders, and support managers to track improvement over time.

See `templates/support-audit-report.md` for the report skeleton.
