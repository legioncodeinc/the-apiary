# Reports

This folder accumulates past audit reports and coaching outputs generated by `agile-scrum-worker-bee`.

## File naming convention

```
YYYY-MM-DD-<team-or-topic>-<report-type>.md
```

Examples:
- `2026-05-20-acme-team-scrum-audit.md`
- `2026-06-01-startup-founders-dod-session.md`

## Report types

- **scrum-audit** — Full process audit using `templates/scrum-audit-report.md`
- **dod-session** — Definition of Done authoring or review session
- **retro-facilitation** — Retrospective facilitation output
- **estimation-coaching** — Estimation technique selection and coaching session

## Retention

Reports in this folder are informational. They are not versioned alongside the stinger's procedure files. Archive or delete reports older than 12 months if they no longer serve the team's learning.
