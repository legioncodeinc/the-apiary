# Reports

This folder collects past tour audit reports generated by `product-tour-onboarding-ui-worker-bee`.

Each report is a dated markdown file following the shape defined in `templates/tour-audit-report.md`.

Naming convention: `YYYY-MM-DD-{scope}-tour-audit.md`

Examples:
- `2026-05-20-dashboard-onboarding-tour-audit.md`
- `2026-06-01-full-product-tour-health-audit.md`

Reports accumulate over time as the product evolves. They form the audit trail that maintenance reviews reference.
