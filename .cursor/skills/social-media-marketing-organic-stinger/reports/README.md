# Reports

This folder accumulates past social media audit reports produced by `social-media-marketing-organic-worker-bee`.

Each report is a completed instance of `templates/social-audit-report.md` for a specific founder or product. Reports are named:

```
YYYY-MM-DD-[product-or-founder-handle]-audit.md
```

Reports are confidential by default and should not include customer names or identifiable information without explicit permission.

---

## Report types

| Type | Description |
|---|---|
| Social presence audit | Full anti-pattern + voice + platform analysis for a new founder |
| Calendar review | Review of an in-progress 4-week calendar against benchmarks |
| 90-day check-in | Engagement rate and trajectory review at the 90-day mark |
| Platform migration | Analysis when a founder is moving from one primary platform to another |

---

## Quality bar

A good audit report:
- Contains specific data (follower counts, ER, post counts) not just qualitative observations
- Lists all 10 anti-patterns by name with Y/N status
- Provides a before/after post rewrite example
- Ends with 3 prioritized remediation actions and a growth expectations summary

A thin audit report that lacks these elements should be rejected and redone.
