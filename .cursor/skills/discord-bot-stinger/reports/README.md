# Reports

This folder collects past audit reports and bot review outputs produced by `discord-bot-worker-bee`.

Each report follows the template at `../templates/audit-report.md`.

Reports are named: `YYYY-MM-DD-<bot-name>-audit.md`

The folder is initially empty; reports accumulate over time as the Bee is invoked on real projects.

For a reference template, see `../templates/audit-report.md`.

For audit methodology, see `../guides/` (all guides are relevant to the audit workflow).
