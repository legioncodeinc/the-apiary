# Reports

This folder collects past CHANGELOG audit reports produced by `changelog-release-notes-worker-bee` when invoked in audit mode.

Audit reports are placed here at: `reports/<YYYY-MM-DD>-changelog-audit.md`

The audit rubric lives in `guides/05-audit-playbook.md`. The worked example lives at `examples/audit-report-example.md`.

---

*This folder is initially empty. Reports accumulate over time as audits are run.*
