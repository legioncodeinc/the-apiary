# Reports

This folder accumulates past audit reports and decision memos generated by `hr-payroll-worker-bee`.

## Naming convention

```
YYYY-MM-DD-<company-slug>-<type>.md
```

Examples:
- `2026-05-20-synthflow-platform-selection.md`
- `2026-06-15-acme-classification-audit.md`
- `2026-07-01-widgetco-eor-evaluation.md`

## Report types

| Type | Source template | Description |
|---|---|---|
| `platform-selection` | `templates/decision-memo.md` | Platform recommendation memo |
| `classification-audit` | `templates/audit-checklist.md` | Worker classification compliance audit |
| `eor-evaluation` | `templates/decision-memo.md` | EOR platform recommendation |
| `migration-plan` | `guides/07-migration-playbook.md` | Platform migration planning output |
| `compliance-audit` | `templates/audit-checklist.md` | Full HR/payroll compliance audit |

No reports yet. The first report will be filed when hr-payroll-worker-bee completes its first persistent audit run.
