# Reports

This folder collects cron audit reports generated by `cron-scheduling-worker-bee` over time.

Each audit report is named `YYYY-MM-DD-<project-slug>-cron-audit.md` and follows the structure from `guides/06-audit-and-inventory.md`:

1. Executive summary (total jobs, risk distribution, top 3 recommendations)
2. Risk assessment matrix (per-job table)
3. Per-job specs (from `templates/cron-job-spec.md`)
4. Prioritized action plan

The folder is initially empty. Reports accumulate here as audits are run.
