# ICP Definition Worksheet

Complete this worksheet before building any list or writing any sequence. A precise ICP reduces wasted sends, improves reply rate, and produces signal you can learn from.

Reference: `guides/05-list-hygiene.md`

## Section 5: The problem statement test

**The one problem your product solves for this ICP:**
```
In one sentence: [job title] at [company type] struggles with _____.
```

**Validation test — answer honestly:**
- Would this problem statement resonate with 7 out of 10 contacts who match the ICP description above? [ ] Yes [ ] No
- Can you name 3 companies that match this ICP and confirm they actually have this problem? [ ] Yes [ ] No
- Has anyone in this ICP described this problem in their own words publicly? [ ] Yes (where: _____) [ ] No

If any answer is No: the ICP or problem statement needs more validation before running a campaign.

---

## Section 6: ICP summary (one sentence)

Fill this in after completing all sections above:

```
We sell to [job title] at [company type, size, geography] who are [buying trigger].
```

**This sentence should be specific enough that someone could pull a list in Apollo from it alone.**
