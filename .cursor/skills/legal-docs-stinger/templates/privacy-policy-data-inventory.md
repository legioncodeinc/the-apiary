## Checklist: special category data

Special category data (GDPR Art. 9) requires explicit consent or a specific legal basis. Check if any of the following apply:

- [ ] Health or medical data
- [ ] Biometric data (fingerprints, facial recognition)
- [ ] Genetic data
- [ ] Racial or ethnic origin
- [ ] Political opinions, religious beliefs
- [ ] Trade union membership
- [ ] Sexual orientation or gender identity
- [ ] Criminal convictions

If any box is checked, stop: this requires an attorney to draft the privacy policy for this category.
