## Maintenance instructions

1. When adding a new sub-processor: add a row, set effective date (30 days from now for advance notice), send notification to mailing list.
2. When removing a sub-processor: add a change log row, update effective date.
3. When a sub-processor loses DPF certification: immediately check whether SCCs are needed; update DPA if required.
4. Re-verify DPF certifications annually at https://www.dataprivacyframework.gov/list.
