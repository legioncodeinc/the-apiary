# design-system-stinger

Cursor IDE skill that equips the `design-system-worker-bee` Bee with the
procedure for bootstrapping a complete design system from scratch for
any product. The skill encodes the canonical seven-artifact folder
structure (master brief, tokens CSS, utility CSS, per-component briefs,
per-screen briefs, static HTML examples, README), a library of aesthetic
starter kits, and the interview procedure that extracts the aesthetic
from the user.

Paired Bee: `design-system-worker-bee`.
Companion Bee (after handoff): `ux-ui-worker-bee`.
